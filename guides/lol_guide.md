# League of Legends - Руководство по привязке аккаунта

## Как найти свой Riot ID:

### Метод 1: В игре
1. Запустите League of Legends
2. В правом верхнем углу нажмите на свой иконку
3. Ваш Riot ID отобразится в формате: `Имя#Тег`
   Пример: `ProPlayer#EUW`

### Метод 2: Через клиент Riot
1. Откройте клиент Riot Games
2. Нажмите на свой аватар
3. Выберите "Настройки учетной записи"
4. Ваш Riot ID будет указан

## Регионы LoL:

### Доступные регионы:
- **EUW** (Европа Запад): `euw1`
- **EUNE** (Европа Северо-Восток): `eun1`
- **NA** (Северная Америка): `na1`
- **KR** (Корея): `kr`
- **BR** (Бразилия): `br1`
- **LAN** (Латинская Америка Север): `la1`
- **LAS** (Латинская Америка Юг): `la2`
- **OCE** (Океания): `oc1`
- **RU** (Россия): `ru`
- **TR** (Турция): `tr1`
- **JP** (Япония): `jp1`

## Что будет отслеживаться:

### Основная статистика:
- K/D/A соотношение
- CS (количество добитых миньонов)
- CS per minute
- Золото и опыт
- Очки зрения
- Урон чемпионам
- Урон по башням

### Детальная аналитика:
- Win rate по чемпионам
- Статистика по линиям
- Эффективность сборок
- Время активации способностей
- Участие в командных боях
- Объективный контроль

### AI рекомендации:
- Выбор чемпионов под вашу игру
- Оптимальные сборки
- Тактические советы
- Улучшение фарма
- Управление видением
