# Valorant - 绑定账户指南

## 如何找到您的 Valorant 账户信息

### 所需信息：
1. **Riot ID**: 您的游戏用户名（例如：Player#1234）
2. **Tag Line**: #后面的部分（例如：#1234）

### 查找步骤：
1. 启动 Valorant 游戏
2. 进入主菜单
3. 点击右上角的设置图标
4. 在"常规"选项卡中查看您的 Riot ID
5. ID 格式为：用户名#标签

### 示例：
- Riot ID: `ProPlayer#EUW`
- 用户名: `ProPlayer`
- 标签: `EUW`

## 重要说明：
- 确保您的账户是公开的
- 需要启用游戏历史记录
- 区域选择非常重要（NA, EU, APAC 等）

## 绑定后您可以获得：
- 实时比赛统计
- K/D 比率跟踪
- ACS（战斗得分）分析
- 经济管理数据
- 特工表现统计
